# CobbleDistrict

Simple PaperMC plugin for 1.21 that adds:
- `/setdistrict` (OP only) – set spawn point
- `/district` – instant teleport to set district

## Features
- Saves location in config.yml
- Permission: `cobbledistrict.tp` (default true)

## Build
```bash
mvn clean package

## Init
Install the latest release in the [releases](https://github.com/kebabmario/CobbleDistrict)
Bring the .jar file into the plugins folder of your Minecraft server.
